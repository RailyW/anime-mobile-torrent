# 文档索引

本目录保存项目调研、架构候选、阶段计划和后续实现说明。

## 目录说明

- [research](research/README.md)：保存外部 API、第三方库、开源项目和 Android 平台能力调研记录。
- [architecture](architecture/README.md)：保存架构候选方案、阶段路线、关键决策和待确认事项。

## 维护规则

1. 当新增调研结论、技术选型或架构决策时，应同步更新本索引或对应子目录 README。
2. 当后续创建代码模块时，每个独立模块都需要在模块目录内提供 README，说明模块职责、文件结构、关键接口和依赖。
3. 文档中的外部结论应尽量附来源链接，避免只保留口头判断。
